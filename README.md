<h1> 🚗 Mech System </h1>
Implementation of 3 RESTful services for Mech System MVP. </br>

<h3> Endpoints </h3>
<ul>
  <li>Post</li> Register a mechanic.
  <li>GetAll</li> List all registered mechanics.
  <li>Get</li> Search for specific mechanics.
</ul>
